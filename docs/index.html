Calendrier liturgique catholique

Les calendriers classiques omettent souvent les grandes fêtes et solennités de l'Église catholique. Ce calendrier remédie à cela.

Ouvrir le calendrier

Google Agenda

Étape 1 : Ajouter le calendrier depuis le web
  • Ouvrez votre navigateur et allez sur Google Agenda.
  • Si vous êtes sur téléphone, en bas, à côté de Afficher, cliquez sur Bureau pour afficher la version pour ordinateur.
  • Appuyez sur le menu en hamburger en haut à gauche ≡.
  • Dans la colonne de gauche, à côté de Autres agendas, appuyez sur +.
  • Sélectionnez À partir de l'URL.
  • Collez le lien du calendrier puis appuyez sur Ajouter l'agenda.
Étape 2 : Activer l'agenda dans l'application mobile
  • Ouvrez l'application Google Agenda.
  • Appuyez sur le menu en hamburger en haut à gauche ≡, puis Paramètres.
  • Touchez dans la liste le nouvel agenda pour qu'il apparaisse dans votre calendrier.

iPhone (Calendrier Apple)

Solution 1 : Depuis Réglages
  • Allez dans Réglages > Calendrier > Comptes Calendrier.
  • Appuyez sur Ajouter un compte > Ajouter un autre compte... > Calendrier (abonnement).
  • Collez le lien du calendrier, puis touchez Suivant et Enregistrer.
Solution 2 : Depuis un navigateur
  • Ouvrez le lien sur votre téléphone.
  • Quand la fenêtre s'affiche, touchez Continuer, puis Tout ajouter.
Attention : Cette méthode ne synchronise pas les futures mises à jour du calendrier.
